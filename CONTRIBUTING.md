# データフィードバック・ガイドライン

sagebase のデータ品質向上にご協力いただきありがとうございます。

## 報告の種類

### データ修正の報告

データに誤りを見つけた場合にご利用ください。

**報告に含めていただきたい情報:**

1. **sagebase_id**（わかる場合）: 対象レコードの識別子
2. **データ種別**: 政治家 / 政党 / 議案 / 発言 / 選挙 / 会派 / その他
3. **現在の値**: 現在表示されている誤った内容
4. **正しい値**: 修正後の期待値
5. **根拠URL**: 公式サイト等のエビデンス（任意ですが、あると対応が迅速になります）

### データ追加のリクエスト

sagebase にまだ収録されていないデータの追加をリクエストできます。

**リクエストに含めていただきたい情報:**

1. **対象議会・選挙**: どの議会、どの選挙のデータか
2. **リクエスト内容**: 追加してほしいデータの説明
3. **データソース**: 公式サイトURL等（任意）

## 対応フロー

1. Issueを作成していただくと、運営チームが確認します
2. 確認後、対応を開始する際にコメントでお知らせします
3. データ修正・追加が完了したらIssueをクローズします

## 注意事項

- 報告は日本語または英語で受け付けています
- 1つのIssueにつき1つの報告をお願いします
- 重複報告を避けるため、既存のIssueを事前にご確認ください
- 個人情報や非公開情報の投稿はお控えください。公開されている情報のみを根拠としてください

## sagebase_id の確認方法

[sage-base.com](https://sage-base.com) 上のデータ詳細ページ、または BigQuery 上のレコードで `sagebase_id` カラムを確認できます。
